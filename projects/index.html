Projects

  • Temporal Random Indexing (TRI): A system for the analysis of meaning variations over time, described in [1]. Available under “GNU GENERAL PUBLIC LICENSE Version 3.
  • Time-Aware Information Retrieval (TAIR): A system for Information Retrieval that enables the expression of temporal constraints, described in [2]. Available under “GNU GENERAL PUBLIC LICENSE Version 3”.
  • Lesk Word Sense Disambiguation Distributional Semantic Model (Lesk-WSD-DSM): A Lesk-based disambiguation Algorithm, described in [3]. Available under “GNU GENERAL PUBLIC LICENSE Version 3”.
  • Quantum Information Retrieval framework (QIR) I contributed to the quantum query algebra for query representation, described in [4].
  • SEmantic N-levels Search Engine (SENSE) Described in [5].
